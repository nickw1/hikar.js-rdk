# hikar.js rewrite using LocAR.js and RDK

This is a new version of [hikar.js](https://github.com/nickw1/hikar.js), to be rewritten using [LocAR.js](https://github.com/AR-js-org/locar.js) and [RDK](https://github.com/omnidotdev/RDK).

Currently it has limited functionality and is early in development, but it is the actively-developed version.
